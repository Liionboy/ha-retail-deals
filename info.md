## 🛒 Retail Deals Romania

Integrare Home Assistant pentru colectarea reducerilor de la magazinele din România.

### Magazine suportate
- **Auchan** — API direct (VTEX)
- **Kaufland** — via PromoAzi.ro
- **Lidl** — via PromoAzi.ro  
- **Carrefour** — via PromoAzi.ro

### Funcționalități
- Colectare automată la interval configurabil
- Senzori pentru top reduceri, per magazin, și individuali
- Configurare din UI (config_flow)
- Filtrare după discount minim
- Notificări automate la reduceri mari

### Instalare
1. Instalează prin HACS
2. Restart Home Assistant
3. Adaugă integrarea din Settings → Devices & Services
